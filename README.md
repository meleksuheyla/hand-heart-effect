# hand-heart-effect
# ❤️ Hand Detection with Heart Effect

This project was created using Python, OpenCV, and MediaPipe.

When a hand is detected via the camera, hearts appear in random locations on the screen.

## 🚀 Technologies Used

- Python
- OpenCV
- MediaPipe

## 📷 Features

- Real-time camera view
- Hand detection system
- Random heart effect on screen
- Exit with ESC key
